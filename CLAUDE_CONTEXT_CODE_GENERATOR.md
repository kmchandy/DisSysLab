# DisSysLab — Code Generator

## What the Code Generator Does

Takes the confirmed JSON from the role parser and office parser and
generates a complete runnable DisSysLab `app.py`.

Input:
- `roles_json`: list of parsed role dicts (from role parser)
- `office_json`: parsed office dict (from office parser)

Output:
- `app.py`: complete runnable DisSysLab app
- `test_app.py`: structural tests

---

## How Role JSON Maps to a Role Function

Each role becomes:
1. A system prompt string built from the role's job + persistent state + routing rules
2. A Python function that calls `ai_agent` with that prompt
3. A `Role` node wrapping that function

### System Prompt Construction

The system prompt for a role is built from the role JSON as follows:

```
{job}

{for each persistent_state field:}
The message has a field called "{field}" whose value is a {type}.
If absent, treat as {default}.

{for each routing_rule:}
If {condition}, include "{send_to[0]}" (and "{send_to[1]}" if fanout)
in the "send_to" list in your response.
{if modify is not null:} Also {modify}.

Return JSON only, no explanation, no nested JSON:
{"send_to": ["channel1"], "{field1}": value, ...}
```

The `send_to` field in the response is a list of channel names.
The role function reads this list and routes accordingly.

### Role Function Pattern

```python
_{role_name}_prompt = """..."""
_{role_name}_ai = ai_agent(_{role_name}_prompt)

def {role_name}_fn(msg):
    import json, re
    text = json.dumps(msg) if isinstance(msg, dict) else str(msg)
    try:
        raw    = _{role_name}_ai(text)
        match  = re.search(r'\{.*\}', raw, re.DOTALL)
        result = json.loads(match.group())
        out    = {**msg, **result}
        destinations = result.get("send_to", ["{default_channel}"])
        return [(out, dest) for dest in destinations]
    except Exception as e:
        print(f"[{role_name}] Error: {e}")
        return []
```

### Role Node

```python
{role_name} = Role(
    fn={role_name}_fn,
    statuses={sends_to},
    name="{role_name}"
)
```

## How Office JSON Maps to a Network

### Agent Instances

Each agent in the office becomes a Role node using its role's function:

```python
# Agent: Susan is an editor
Susan = Role(fn=editor_fn, statuses=["copywriter", "archivist"], name="Susan")

# Agent: Anna is a writer
Anna = Role(fn=writer_fn, statuses=["client"], name="Anna")
```

### Source Instances

```python
from components.sources.rss_normalizer import al_jazeera, bbc_world, npr_news

_al_jazeera = al_jazeera(max_articles=5)
src_al_jazeera = Source(fn=_al_jazeera.run, name="al_jazeera")
```

### Sink Instances

```python
from components.sinks.sink_jsonl_recorder import JSONLRecorder
from components.sinks.console_display import ConsoleDisplay

_recorder = JSONLRecorder(path="editorial_output.jsonl", mode="w", flush_every=1)
jsonl_recorder = Sink(fn=_recorder, name="jsonl_recorder")

_display = ConsoleDisplay(verbose=False)
console_printer = Sink(fn=_display, name="console_printer")
```

### Network Connections

Each connection maps to a network edge using explicit port syntax:

```python
# Source to agent: al_jazeera's destination is Susan
(src_al_jazeera, Susan),

# Agent channel to agent: Susan's copywriter is Anna
# copywriter is index 0 in Susan's statuses ["copywriter", "archivist"]
(Susan.out_0, Anna),

# Agent channel to sink (fanout): Susan's archivist are jsonl_recorder and console_printer
# archivist is index 1 in Susan's statuses
(Susan.out_1, jsonl_recorder),
(Susan.out_1, console_printer),

# Feedback loop: Anna's client is Susan
# client is index 0 in Anna's statuses ["client"]
(Anna.out_0, Susan),
```

Port index = position of channel name in `sends_to` list from role JSON.

## Complete Generated app.py Example

Input: editor + writer roles, news_editorial office.

```python
# app.py — AUTO-GENERATED by office_compiler.py
# Office: news_editorial
# Roles: editor, writer
#
# Routing:
#   al_jazeera  →  Susan
#   bbc_world   →  Susan
#   npr_news    →  Susan
#   Susan [copywriter]  →  Anna
#   Susan [archivist]   →  jsonl_recorder, console_printer
#   Anna  [client]      →  Susan

import json
import re

from dsl import network
from dsl.blocks import Source, Sink
from dsl.blocks.role import Role
from components.transformers.ai_agent import ai_agent
from components.sources.rss_normalizer import al_jazeera, bbc_world, npr_news
from components.sinks.sink_jsonl_recorder import JSONLRecorder
from components.sinks.console_display import ConsoleDisplay


# ── Role: editor ──────────────────────────────────────────────────────────────

_editor_prompt = """analyze the sentiment of each article and score it from 0.0 (most negative) to 1.0 (most positive).

The message has a field called "rewrites" whose value is a number. If absent, treat as 0.

If score < 0.25 or score > 0.75, include "archivist" in the send_to list.
If 0.25 <= score <= 0.75 and rewrites < 3, include "copywriter" in the send_to list. Also add 1 to rewrites.
If 0.25 <= score <= 0.75 and rewrites >= 3, include "archivist" in the send_to list.

Return JSON only, no explanation, no nested JSON:
{"send_to": ["channel"], "score": <float>, "rewrites": <int>}"""

_editor_ai = ai_agent(_editor_prompt)


def editor_fn(msg):
    text = json.dumps(msg) if isinstance(msg, dict) else str(msg)
    try:
        raw    = _editor_ai(text)
        match  = re.search(r'\{.*\}', raw, re.DOTALL)
        result = json.loads(match.group())
        out    = {**msg, **result}
        destinations = result.get("send_to", ["archivist"])
        return [(out, dest) for dest in destinations]
    except Exception as e:
        print(f"[editor] Error: {e}")
        return []


# ── Role: writer ──────────────────────────────────────────────────────────────

_writer_prompt = """Rewrite the article to be more strongly opinionated — push the sentiment
either more positive or more negative, whichever requires less change.
Preserve all fields from the input message.

Always include "client" in the send_to list.

Return JSON only, no explanation, no nested JSON:
{"send_to": ["client"], "text": "<rewritten text>"}"""

_writer_ai = ai_agent(_writer_prompt)


def writer_fn(msg):
    text = json.dumps(msg) if isinstance(msg, dict) else str(msg)
    try:
        raw    = _writer_ai(text)
        match  = re.search(r'\{.*\}', raw, re.DOTALL)
        result = json.loads(match.group())
        out    = {**msg, **result}
        destinations = result.get("send_to", ["client"])
        return [(out, dest) for dest in destinations]
    except Exception as e:
        print(f"[writer] Error: {e}")
        return []


# ── Sinks ─────────────────────────────────────────────────────────────────────

_recorder = JSONLRecorder(path="editorial_output.jsonl", mode="w", flush_every=1)
jsonl_recorder  = Sink(fn=_recorder, name="jsonl_recorder")

_display = ConsoleDisplay(verbose=False)
console_printer = Sink(fn=_display,   name="console_printer")


# ── Sources ───────────────────────────────────────────────────────────────────

_al_jazeera = al_jazeera(max_articles=5)
_bbc_world  = bbc_world(max_articles=5)
_npr_news   = npr_news(max_articles=5)

src_al_jazeera = Source(fn=_al_jazeera.run, name="al_jazeera")
src_bbc_world  = Source(fn=_bbc_world.run,  name="bbc_world")
src_npr_news   = Source(fn=_npr_news.run,   name="npr_news")


# ── Agents ────────────────────────────────────────────────────────────────────

Susan = Role(fn=editor_fn, statuses=["copywriter", "archivist"], name="Susan")
Anna  = Role(fn=writer_fn, statuses=["client"],                  name="Anna")


# ── Network ───────────────────────────────────────────────────────────────────

g = network([
    (src_al_jazeera, Susan),
    (src_bbc_world,  Susan),
    (src_npr_news,   Susan),
    (Susan.out_0, Anna),              # copywriter → Anna
    (Susan.out_1, jsonl_recorder),    # archivist  → jsonl (fanout)
    (Susan.out_1, console_printer),   # archivist  → console (fanout)
    (Anna.out_0,  Susan),             # client     → Susan (feedback loop)
])


# ── Run ───────────────────────────────────────────────────────────────────────

if __name__ == "__main__":
    print()
    print("News Editorial Organization")
    print("=" * 60)
    print()
    g.run_network()
    _recorder.finalize()
    _display.finalize()
    print()
    print("Done!")
    print()
```

## Code Generation Rules

1. Role functions always use `ai_agent` — never `demo_ai_agent`.

2. The system prompt is built from the role JSON — job + persistent state
   + routing rules expressed as instructions to include channel names in
   the `send_to` list.

3. Every role function returns `[(out, dest) for dest in destinations]`
   where destinations comes from `result.get("send_to", [default])`.

4. Agent variable names are the agent names from the office spec (Susan, Anna).
   Source variable names are prefixed with `src_` to avoid collision with
   factory function names.

5. Port indices follow the order of `sends_to` in the role JSON:
   index 0 = first channel, index 1 = second channel, etc.

6. Always include `if __name__ == "__main__":` guard.

7. Always call `finalize()` on sinks that have it.

8. Generated test imports use full dotted module path derived by walking
   `__init__.py` files up to the DisSysLab root.
