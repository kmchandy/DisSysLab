== NOTES ON DESIGN PHILOSOPHY ==

"Same pattern - different app" is the core pedagogical goal for M06→M07.
Network wiring is identical. Students see the gather-scatter pattern
is universal, not domain-specific.

Hobbyist arc: M07 photo scorer → M08 human labels photos → M08 RL learns
from feedback. This is RLHF (same concept as ChatGPT training) built from
components students already understand.

The download_demo_images.py script is deliberately the first customization
point. Students edit it to fetch photos they care about before touching
any analyzer code. This follows the "excitement first" pedagogy.

Social media readiness checker (M07 option 4) is offered as homework
in the README - same technical implementation, different framing.
